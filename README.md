# DeepLearning
Modelo de deep learning para generar texto.

Entrenado con secuencias de texto de Wikipedia, diálogos de South Park, y secuencias de texto de conversaciones entre humanos y entre humanos-chatbots.

Correr usando la interfaz de texto al final del notebook.
